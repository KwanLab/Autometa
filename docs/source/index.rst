Autometa documentation
====================================

**Autometa: automated extraction of microbial genomes from individual shotgun metagenomes**

An automated binning pipeline for single metagenomes, in particular host-associated and highly complex ones. Autometa is copyright 2020 Ian J. Miller, Evan R. Rees, Izaak Miller and Jason C. Kwan, and is released under the GNU Affero General Public License v3 (see License). If you find Autometa useful to your work, please cite our [Autometa]_ paper.


Quickstart
^^^^^^^^^^

Run with below code::

    python autometa.py metagenome.config
